# Distributed Systems
CIS 630, Spring 2017, University of Oregon

This repository contains the two projects and survey paper developed for the Distributed Systems course (CIS 630).

## Contents

* ./project1 - Distributed implementation of the page-rank algorithm in C++ using the MPI Library
* ./project2 - Development of message server based on a ring buffer. Implemented using multiple processes and UDP socket-based communication, where processes can dynamically join and leave the ring buffer
* ./paper - Survey paper on MPI+X alternatives for heterogeneous distributed computing
./README.md - This file

## Built With
* [C++]() - Language used in project development
* [MPI](https://en.wikipedia.org/wiki/Message_Passing_Interface) - MPI library used to implement project1/part2.
